Desarrollo Agil con BDD en Java

09/07/2013 · Lectura de 1 minutos · Comentarios
BDDEventosAgilidadJava

Continuando mi tour por las comunidades de desarrolladores de Bogotá, hice presencia en el 7mo. Java Day para compartir con la comunidad de Java mi charla de Desarrollo Ágil con BDD.

El Java Day, efectuado el sábado 6 de Julio de 2013, contó con dos ponencias, la primera referente a la arquitectura, sus estilos más usados y su importancia. La charla estuvo a cargo de David Fernando de la Peña, y partícularmente me gusto mucho su analogía con la arquitectura tradicional, y como dependiendo del tamaño y complejidad del sistema, el trabajo a realizar en esta área debe ser distinto.

Luego, llevé a cabo mi charla, intercambiando con los asistentes, quienes una buena parte (el equipo de Bosz Digital, patrocinantes del Java Day) trabajan y conocen las metodologías ágiles.
